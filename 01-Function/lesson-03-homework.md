# Homework 3: Increasing, Decreasing, Even, and Odd Functions

## Instructions

Show your work clearly. Use interval notation whenever possible.

## Part A: Definitions and Basic Classification

1. In your own words, explain the difference between an increasing function and a decreasing function.

2. Complete the rigorous definitions.

- A function is increasing on an interval if whenever $x_1<x_2$, then __________.
- A function is decreasing on an interval if whenever $x_1<x_2$, then __________.

3. Decide whether each function is increasing, decreasing, constant, or neither on all real numbers.

- $f(x)=7x-5$
- $g(x)=12-4x$
- $h(x)=3$
- $p(x)=x^2+1$

4. The table below gives values of a function $f$.

| $x$ | $-3$ | $-1$ | $0$ | $2$ | $4$ |
|---|---|---|---|---|---|
| $f(x)$ | $8$ | $5$ | $3$ | $0$ | $-2$ |

Using only the table, decide whether the function appears to be increasing or decreasing on the listed inputs.

## Part B: Intervals of Increase and Decrease

- the interval where the function is increasing

6. Consider

8. Consider

$$
p(x)=\sin x
$$

- where the function is decreasing

9. Consider the piecewise function

$$

\begin{cases}
2-x, & x<1 \\
4, & 1\le x<3 \\
x-1, & x\ge 3
\end{cases}
$$

Describe where the function is:

- decreasing
- constant
- increasing

## Part C: Even, Odd, or Neither

11. Determine whether each function is even, odd, or neither.

$$
