# Homework 1: Introduction to Polynomials

## Instructions

Show your work clearly. If a function is **not** a polynomial, explain which rule it breaks. If a function **is** a polynomial, give its degree when asked.

## Part A: Definition and Basic Classification

1. In your own words, define a polynomial function.

2. Decide whether each function is a polynomial. Briefly explain your answer.

- $f(x)=3x^4-2x+7$
- $g(x)=x^{-1}+5$
- $h(x)=8$
- $p(x)=x^{1/2}+1$
- $q(x)=6x^2-4x+\dfrac{1}{3}$

3. Decide whether each function is a polynomial. If it is a polynomial, state its degree.

- $f(x)=\dfrac{x^2+x+5}{\pi}$
- $g(x)=\dfrac{1}{x+1}$
- $h(x)=\sqrt{x}+3$
- $p(x)=\dfrac{3x^5-2x+1}{4}$
- $q(x)=\sin x$

4. Explain why a constant denominator is allowed in a polynomial, but a denominator containing $x$ is not.

## Part B: Terms, Coefficients, and Constant Term

5. For

$$
f(x)=-4x^5+3x^2-x+9
$$

identify:

- the terms
- the coefficient of $x^5$
- the coefficient of $x$
- the constant term

6. For

$$
g(x)=7-2x^3+x^4
$$

identify:

- the terms
- the coefficient of $x^4$
- the coefficient of $x^3$
- the constant term

## Part C: Standard Form and Degree

7. Write each polynomial in standard form. Then state the degree and leading coefficient.

- $f(x)=5-x^3+2x^5-x$
- $g(x)=9+4x^2$
- $h(x)=-7x+x^6-3x^2+1$

8. Each expression below is a polynomial, but it is **not** written in standard form. Find the degree.

- $f(x)=3x-x^7+2$
- $g(x)=8+x^2-x^5+4x$
- $h(x)=1-6x^3+x^2$
- $p(x)=x+x^9-x^4+3$

9. Without rewriting fully in standard form, find the degree of each polynomial.

- $f(x)=12-5x+x^{11}-3x^4$
- $g(x)=\dfrac{2}{3}x^6+9-x^2$
- $h(x)=4x^3-7+2x^8-x$
- $p(x)=6$

10. For each polynomial below, state:

- whether it is in standard form
- its degree

- $f(x)=2x^4-x^2+3x-1$
- $g(x)=7+x^3-2x^5$
- $h(x)=-9x+4x^2+1$
- $p(x)=5$

## Part D: Why It Is Not a Polynomial

11. State the reason each function is **not** a polynomial.

- $f(x)=x^{-3}+2x$
- $g(x)=\sqrt{x}-1$
- $h(x)=\cos x$
- $p(x)=\dfrac{x+2}{x-5}$

12. Rewrite each expression in a form that makes it easier to see why it is not a polynomial.

- $\sqrt{x}$
- $\dfrac{1}{x^2}$
- $\dfrac{3}{x}$

Then state the reason in each case.

## Part E: Mixed Practice

13. For each function:

- decide whether it is a polynomial
- if yes, give the degree
- if no, explain why not

- $f(x)=0$
- $g(x)=\dfrac{x^3-2x+1}{5}$
- $h(x)=x+\sqrt{x}$
- $p(x)=4x^2-\dfrac{7}{2}x+1$
- $q(x)=\dfrac{x^2+1}{x}$

14. One student says, "Any function with a fraction is not a polynomial." Is this correct? Explain using at least one example.

15. One student says, "If a polynomial is not in standard form, then you cannot tell its degree." Is this correct? Explain using at least one example.
